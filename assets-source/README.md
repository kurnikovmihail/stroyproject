# assets-source

Служебная папка с исходными медиа, которые не участвуют напрямую в сборке сайта.

Структура:
- `raw-imports/` — исходники и импортированные материалы из рабочего процесса.
- `public-unused/` — файлы, которые были в `public`, но сейчас не используются в `src`.

Рабочие ассеты сайта находятся в `public/images`.
